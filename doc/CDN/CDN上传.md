



#### 简要描述

- 上传CDN，获取fileid。

#### 请求URL
- ` http://127.0.0.1:8888/api/`

#### 请求方式
- POST 

#### 参数

| 参数名      | 必选 | 类型     | 说明                        |   
|:---------|:---|:-------|---------------------------|   
| type     | 是  | int    | 接口编号                      |   
| filePath | 是  | string | 文件绝对路径                    |   
| aeskey   | 是  | string | aeskey                    |   
| fileType | 是  | int    | 文件类型，可参考枚举值，更多类型可从消息XML获取 |   

#### 文件类型

| 文件类型        | 枚举值 |   
|:------------|:----|   
| 图片          | 2   |   
| 视频          | 4   |   
| 文件          | 5   |   
| 大文件(25M及以上) | 7   |   
| 语音          | 15  |   

#### 请求示例

```
 {
  "type": 7,
  "filePath": "C:\\xxx.txt",
  "aeskey": "e742f8ce990980e402b00f89dc8d1d4d",
  "fileType": 5,
 } 

```
