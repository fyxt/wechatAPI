



#### 简要描述

- 发送链接

#### 请求URL
- ` http://127.0.0.1:8888/api/`

#### 请求方式
- POST

#### 参数

| 参数名      | 必选 | 类型     | 说明      |   
|:---------|:---|:-------|---------|   
| type     | 是  | int    | 接口编号    |   
| userName | 是  | string | 接收人wxid |   
| content  | 是  | string | 消息内容    |

| 参数名         | 类型     | 说明   |
